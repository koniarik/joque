◆ joque::task_set

struct joque::task_set

A set of tasks that contains either tasks or another sets.

This forms a tree representing the entire task set.

Convetion is that names inside the structure are concatenated to form a full path. Task x in root set is referred as //x. Task y in subset z is referred as //z/y.
